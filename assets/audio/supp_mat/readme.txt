Supplementary material - StyleWaveGAN: Style-based synthesis of drum sounds with extensive controls using generative adversarial networks

# Folder structure
* brightness_effect : audio samples of generated snare drum with variable brightness target value. Target values are given in the filenames.
* data : audio samples from augmented dataset using extreme values of augmentation parameters, against the original sounds.
* synthesis : audio samples from supervised StyleWaveGAN, 2 per class.

# Abreviations used in filenames
* bd : bass/kick drum
* sd : snare drum
* chh : closed hi-hat
* ohh : open hi-hat
* tom : tom-toms
